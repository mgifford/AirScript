    Quick setup

    1. Open this speaker page on the device with the microphone.
    2. Share the Audience URL or have attendees scan the QR code (both work with no internet).
    3. Click Start Captions and allow microphone permissions.
    4. Audience phones on the same network will receive live captions in real-time.
    5. (Optional) Set a relay URL for advanced SSE updates or third-party transcription.
