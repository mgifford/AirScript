Mac Internet Sharing For AirScript

Use this when venue Wi-Fi blocks local device-to-device traffic. Your Mac will create a local Wi-Fi network so audience phones can open the AirScript demo directly.

Open Speaker Console Open Audience Demo Back To Landing Page

Step By Step (macOS Ventura/Sonoma+)

  1. Open System Settings.
  2. Go to General -> Sharing.
  3. Open Internet Sharing options.
  4. Set Share your connection from to your upstream network (usually Wi-Fi or Ethernet).
  5. Set To computers using to Wi-Fi.
  6. Configure a network name and password under Wi-Fi options.
  7. Toggle Internet Sharing on and confirm.

Then Start AirScript

  1. Open https://mgifford.github.io/AirScript/demo/speaker.html on the presenter laptop.
  2. Join audience phones to the shared network.
  3. Have attendees scan the QR code shown on the speaker page.
  4. Start captions in Chrome or Edge on the speaker device.

If It Still Fails

  • Reconnect the phone to the shared SSID and disable VPN on both devices.
  • Confirm the speaker page shows a valid Audience URL and QR code.
  • If you are using relay mode from GitHub Pages, avoid http:// relay targets from an https:// page.
  • When in doubt, keep the default static mode first, then add relay once baseline works.
